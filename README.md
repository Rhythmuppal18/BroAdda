# 🌐 BroAdda
## Modern Cyber Bistro Experience

> *Fresh vegetarian comfort food meets sleek digital ordering. Welcome to the future of casual dining.*

**[🚀 Live Demo](https://bro-adda-restaurant.vercel.app/) | [📁 GitHub](https://github.com/Rhythmuppal18/BroAdda)**

---

## ✨ What Makes BroAdda Different?

BroAdda isn't just another restaurant website. It's a **premium cyber-bistro** with:

- **32+ Hand-Curated Vegetarian Dishes** — Every item crafted for flavor, texture, and refinement
- **6 Cuisine Moods** — Veg Burgers | Italian Comfort | Asian Favorites | Indian Classics | Dessert Specials | Crafted Beverages
- **Futuristic Terminal Aesthetic** — Sleek cyber-themed design that feels modern, not corporate
- **Smart Digital Menu** — Browse by category, filter by mood, add to cart instantly
- **Seamless Reservations** — Book your table in 30 seconds with a polished form
- **24/7 Digital Service** — Order anytime, anywhere

---

## 🎯 Key Features

### 1. **Digital Menu System**
- Category-based filtering (click a cuisine mood to explore)
- Dish cards with descriptions, pricing, and "Quick Add" buttons
- Search functionality built-in
- Mobile-optimized viewing

### 2. **Smart Ordering**
- Add items to cart without leaving the page
- Real-time subtotal calculation
- Secure payment integration ready
- Order summary before checkout

### 3. **Table Reservations**
- Quick reservation form (Name, Email, Party Size, Time)
- Dropdown selectors for guest count (1-10+ guests)
- Time slot picker
- Confirmation system

### 4. **Modern Cyber-Bistro Branding**
- Terminal-inspired navigation header
- Hero section with powerful CTAs
- "Why Guests Love BroAdda" social proof section
- System-themed footer with operations manifest

## 💻 How It Works

### Homepage Flow

```
HEADER (Logo + Navigation)
    ↓
HERO SECTION (Tagline + CTAs)
    ↓
CHEF'S SPECIAL (Feature dish with image)
    ↓
MENU CATEGORIES (6 selectable moods)
    ↓
MENU GRID (32+ dishes with prices & quick add)
    ↓
SOCIAL PROOF (Why guests love us)
    ↓
RESERVATION FORM (Book your table)
    ↓
FOOTER (Branding + operations terminal theme)
```

### JavaScript Features

**Menu Filtering:**
```javascript
Click cuisine category → Filter dishes → Show matching items
```

**Add to Cart:**
```javascript
Click "Quick Add" → Item added to cart → Subtotal updates in real-time
```

**Reservations:**
```javascript
Fill form → Validate inputs → Submit booking → Confirmation
```

## 🚀 Quick Start

### View Live
Just visit: **[https://bro-adda-restaurant.vercel.app/](https://bro-adda-restaurant.vercel.app/)**

### Run Locally
```bash
# Clone
git clone https://github.com/Rhythmuppal18/BroAdda.git
cd BroAdda

# Open in browser
open index.html
# or drag index.html into your browser
```

### With Live Server (VS Code)
1. Install "Live Server" extension
2. Right-click `index.html`
3. Select "Open with Live Server"
4. Opens at `http://127.0.0.1:5500/`
